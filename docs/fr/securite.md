# Sécurité et conformité

## Authentification

- Accès via identifiants utilisateur.
- Les sessions sont protégées par contrôle d'accès.

## Autorisations

- Les droits sont définis par rôle (Dirigeant, Admin, Gestion dépôt, Technicien).
- Les écrans et actions disponibles varient selon le rôle.

## Traçabilité

- Le module Historique permet de retrouver les mouvements et actions.
