# Documentation FXN ERP (FR)

## Vue d'ensemble

FXN ERP est une plateforme de pilotage des opérations terrain et logistique: dépôts, ressources, interventions, commandes, réservations, facturation et suivi de performance. Cette documentation est destinée aux utilisateurs métier et aux clients (direction, partenaires, donneurs d'ordre).

## Comment utiliser cette documentation

- Commencez par le guide qui correspond à votre profil.
- Consultez ensuite les modules et processus pour les détails opérationnels.
- La FAQ et le glossaire servent de référence rapide.
