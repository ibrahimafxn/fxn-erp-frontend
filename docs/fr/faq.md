# FAQ

## Je ne vois pas un module.

L'accès dépend de votre rôle. Demandez à un admin de vérifier vos permissions.

## Je n'arrive pas à me connecter.

Verifiez vos identifiants et contactez votre admin si besoin.

## Comment retrouver un mouvement ou une action ?

Utilisez le module Historique pour filtrer par date, dépôt ou ressource.

## Comment savoir si un stock est critique ?

Consultez les Alertes stock dans votre espace.
