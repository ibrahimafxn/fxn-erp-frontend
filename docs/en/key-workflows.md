# Key workflows

## Depot management

1. Create or update resource records.
2. Review stock alerts.
3. Track movements through history.

## Resource reservation

1. Review pending reservations.
2. Validate or adjust allocation.
3. Update status and log the action.

## Orders and receipts

1. Create an order.
2. Track processing status.
3. Register related receipts.

## Interventions

1. Import or review interventions.
2. Analyze by technician and period.
3. Track operational indicators.

## Vehicle breakdowns

1. Register a breakdown or incident.
2. Review breakdown history per vehicle.
