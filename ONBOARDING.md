# Onboarding (FXN ERP)

## Prérequis
- Node 18+
- API backend running

## Installation
```bash
npm ci
```

## Démarrage
```bash
ng serve
```

## Démo rapide
1) Charger les données demo (voir `exchange-backend/DEMO.md`).
2) Se connecter avec:
- admin@fxn.local / Admin123
- manager@fxn.local / Manager123

## Vérifications
- Accès par rôle
- Gestion stocks (matériels/consommables)
- Historique des mouvements
